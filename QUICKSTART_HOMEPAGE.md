# 🚀 Quick Start - New Trendyol-Inspired Homepage

## See Your New Homepage in Action

### 1. Navigate to Project
```bash
cd "/Users/doko/Desktop/CODEX/_1_Live Projects/Bangoofrontenddev"
```

### 2. Install Dependencies (if needed)
```bash
yarn install
```

### 3. Start Development Server
```bash
yarn dev
```

### 4. Open in Browser
```
http://localhost:8000
```

## 🔧 Customization

### Update Carousel Slides
Edit: `src/modules/home/components/hero-carousel/index.tsx`
- Change `slides` array (line 8)

### Update Categories
Edit: `src/modules/home/components/category-quick-links/index.tsx`
- Change `quickCategories` array (line 5)

### Update Promotional Banners
Edit: `src/modules/home/components/promotional-banners/index.tsx`
- Change `promoCards` array (line 5)

### Update Brands
Edit: `src/modules/home/components/brand-showcase/index.tsx`
- Change `brands` array (line 5)
